JavaScript

Skryptowy język programowania stosowany głównie do tworzenia aplikacji internetowych. Pozwala na wprowadzanie interakcji poprzez reagowanie na zdarzenia, walidację danych wprowadzanych w formularzach lub tworzenia złożonych efektów wizualnych.

MOŻLIWOŚCI

Javascript to jeden z podstawowych technologii używanych podczas tworzenia stron internetowych. Umożliwia on tworzenie na stronach interaktywnych elementów, a także efektów wizualnych nieosiągalnych samymi kaskadowymi arkuszami stylów oraz walidację danych wprowadzanych przez użytkowników w formularzach. Skrypty te mają znacznie ograniczony dostęp do komputera użytkownika. W języku tym można też pisać pełnoprawne aplikacje przy wykorzystaniu technologii takich jak XUL czy XBL.

FRAMEWORKI

Frameworki JavaScript pozwalają użytkownikom budować skalowalne aplikacje internetowe za pomocą standardowych narzędzi.W przeciwieństwie do bibliotek, frameworki JavaScript definiują projekt aplikacji internetowych, wywołują bibliotekę JS i używają kodu w niej zawartego do ukończenia aplikacji. Oferują one kompletną mapę drogową do budowania aplikacji internetowej zamiast pojedynczego rozwiązania.
